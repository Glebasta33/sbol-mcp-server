##### Описание
Cookbook MCP Server для разработки SBOL Android.
Предоставляет доступ к документации по СБОЛ, который GigaCode может использовать в качестве дополнительного контекста при решении задач.

2 похода:
- **На каждую задачу - отдельную тулзу, которые ходят в свои md (MVP)**
- 2 функции: прочитать список md и прочитать md
  Функции:
- предоставлять доступ к шаблонам решения задач
    - unit-tests
    - data (dto, converters, repo, ...)
    - Каркас (экран + VM) ?
    - ...
- предоставлять доступ к актуальной документации
